# PROYECTO ELABORACION DE PAGINA WEB Maria Carlota - Homemade Mexican Food  
 
 ## Taller de Productividad Basada en Herramientas Tecnologicas

 ### DESCRIPCION DEL PROYECTO
El proyecto consiste en la creación de una página web para el restaurante llamado María Carlota. Este restaurante ofrece una experiencia gastronómica única y tipica de México con técnicas culinarias innovadoras. El objetivo de la página web es proporcionar a los clientes potenciales información sobre el restaurante, su ubicación, menú, horarios de apertura, reservas y eventos especiales.

 ### PROBLEMA IDENTIFICADO
El restaurante María Carlota carece de presencia en línea, lo que dificulta que los clientes potenciales encuentren información relevante sobre el establecimiento. Esto puede llevar a la pérdida de oportunidades de negocio, ya que los clientes pueden optar por otros restaurantes que tengan una presencia en línea más sólida y accesible.

 ### SOLUCION
La solución propuesta es desarrollar una página web atractiva y funcional para el restaurante María Carlota. La página web servirá como una herramienta de marketing digital que permitirá al restaurante llegar a un público más amplio, atraer nuevos clientes y mejorar la experiencia general del cliente al proporcionar información fácilmente accesible y opciones de reserva en línea y no es necesario una excelente conocimiento al manejo de nuestra pagina web.

### INSTALACION DE NetBeans IDE 7.4
A continuacion les comentaré lo que utilicé y lo que hice para tener mi pagina web:

##### 1. Descarga de NetBeans IDE 7.4:
Visita el sitio web oficial de NetBeans o utiliza un motor de búsqueda para encontrar un enlace de descarga confiable para la versión 7.4 del IDE.
##### Selecciona el Paquete de Instalación:
Dependiendo de tu sistema operativo, descarga el paquete de instalación adecuado (por ejemplo, .exe para Windows, .dmg para macOS, .sh o .rpm para Linux).
##### 2. Instala NetBeans:
Ejecuta el archivo de instalación descargado y sigue las instrucciones del asistente de instalación.
Acepta los términos de la licencia y elige la ubicación de instalación si se te pide.
##### 3. Finaliza la Instalación:
Una vez que la instalación haya terminado, puedes optar por lanzar NetBeans IDE de inmediato o hacerlo más tarde desde el acceso directo en el escritorio o el menú de inicio.
### Ejecución:
##### 1. Inicia NetBeans IDE:
Haz doble clic en el acceso directo de NetBeans en el escritorio o busca NetBeans en el menú de inicio y haz clic para abrirlo.
##### 2. Configura el Entorno (Opcional):
Al iniciar NetBeans por primera vez, es posible que se te pida configurar algunas preferencias iniciales, como el tema de la interfaz, los atajos de teclado, etc. Puedes configurar estas opciones según tus preferencias.
### Elaboración de Proyectos:
##### 1. Crea un Nuevo Proyecto:
Una vez que NetBeans esté abierto, puedes crear un nuevo proyecto seleccionando "Archivo" > "Nuevo Proyecto" en la barra de menú.
Selecciona el tipo de proyecto que deseas crear (por ejemplo, "Aplicación Java", "Aplicación Web", etc.) y sigue los pasos del asistente de creación de proyectos.
##### 2. Desarrolla tu Aplicación:
Utiliza el editor de código integrado para escribir tu código fuente.
NetBeans proporciona herramientas de ayuda como resaltado de sintaxis, completado automático de código, depurador y más para facilitar el desarrollo.
##### 3. Ejecuta y Depura tu Aplicación:
Una vez que hayas escrito parte de tu código, puedes ejecutar y depurar tu aplicación seleccionando "Ejecutar" > "Ejecutar Proyecto" o "Depurar Proyecto" en la barra de menú.
##### 4. Guarda tu Proyecto:
Asegúrate de guardar tu proyecto regularmente utilizando la opción "Guardar" o "Guardar Todo" en el menú "Archivo".

Despues de NetBeans en hacer mi pagina web hice mi base de datos que a continuacion sera lo que hice para la elaboracion de mi base de datos:
 
##### 1. Inicia los Servicios de XAMPP:
Abre XAMPP Control Panel.
Inicia los servicios Apache y MySQL haciendo clic en los botones correspondientes. Espera a que los servicios estén "Activos" (los indicadores deben cambiar de "Stop" a "Start" y luego a "Running").
##### 2. Accede a phpMyAdmin:
Abre tu navegador web y visita la siguiente URL: http://localhost/phpmyadmin/.
Esto abrirá la interfaz de phpMyAdmin, que es una herramienta de administración de bases de datos para MySQL que viene incluida en XAMPP.
##### 3. Inicia Sesión en phpMyAdmin:
Por defecto, no hay una contraseña establecida para el usuario "root" en XAMPP, así que simplemente haz clic en "Go" para iniciar sesión.
##### 4. Crea una Nueva Base de Datos:
En la interfaz de phpMyAdmin, verás un panel a la izquierda con una lista de bases de datos (si las hay).
Haz clic en el botón "Nueva" en la parte superior izquierda.
Ingresa un nombre para tu nueva base de datos en el campo "Crear base de datos".
Selecciona la codificación y el cotejamiento (por lo general, puedes dejar estos en sus valores predeterminados).
Haz clic en el botón "Crear".
##### 5. Verifica la Creación de la Base de Datos:
Después de hacer clic en "Crear", deberías ver un mensaje de confirmación de que la base de datos se ha creado correctamente.
En el panel de la izquierda, deberías ver ahora tu nueva base de datos listada.
